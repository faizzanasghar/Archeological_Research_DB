# ArcheoDB - Frontend Client

This is the React-based frontend for the Archaeological Research Database (ArcheoDB), providing a modern, premium interface for researchers and administrators.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

## 🛠️ Key Technologies
- **React 19**: Modern component architecture.
- **Vite**: High-performance development server and bundler.
- **Tailwind CSS 4**: Utility-first styling with advanced design tokens.
- **React Leaflet**: Interactive mapping for excavation sites.
- **Recharts**: Data visualization for research analytics and financials.
- **Zustand**: Lightweight state management for auth and UI states.
- **Framer Motion**: Smooth, high-end micro-animations.

## 🔗 Project Architecture
This frontend communicates with the [ArcheoDB Backend API](../archeo-research-backend).
